개인정보처리방침

시행일: 2026년 6월 8일

모이다(이하 "서비스")는 「개인정보 보호법」 및 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 처리하기 위하여 다음과 같은 개인정보처리방침을 수립·공개합니다.

1. 수집하는 개인정보 항목 및 수집 방법

구분항목수집 방법
필수이메일(소셜 로그인 시 자동), 닉네임소셜 로그인(애플/카카오), 앱 내 입력
선택이름, 성별, 생년월일앱 내 입력
사진·이미지회원이 업로드한 사진 및 이미지 파일카메라 촬영, 사진 라이브러리 선택을 통한 업로드
이용자 생성 콘텐츠피드 게시글(텍스트), 댓글, 투표(설문) 항목 및 응답 데이터앱 내 입력
활동 데이터좋아요·반응 기록, 투표 참여 기록앱 내 활동 시 자동 생성
자동 수집서비스 이용 기록, 기기 정보(OS 버전), 푸시 알림 토큰(FCM)자동 생성

2. 개인정보의 수집·이용 목적

  • 회원 관리: 회원 식별, 소셜 로그인 인증, 프로필 관리
  • 서비스 제공: 모임 운영(회비 관리, 장부, 일정, 공지, 경조사), 푸시 알림 발송
  • 피드·소통 기능: 피드 게시글 작성·표시, 사진 업로드·저장·표시, 댓글 작성·표시, 투표(설문) 생성·참여·결과 표시, 좋아요·반응 기능 제공
  • 서비스 개선: 이용 통계 분석, 오류 해결
  • 콘텐츠 관리: 부적절한 콘텐츠 모니터링 및 이용약관 위반 게시물 관리

3. 기기 접근 권한

서비스는 다음 기기 기능에 대한 접근 권한을 요청할 수 있습니다:

접근 권한목적필수 여부
카메라피드 게시글에 첨부할 사진 촬영선택
사진 라이브러리(갤러리)피드 게시글에 첨부할 이미지 선택선택
알림푸시 알림 수신선택

① 선택 권한은 거부하더라도 해당 기능 외의 서비스 이용에는 제한이 없습니다.

② 접근 권한은 기기의 설정에서 언제든지 변경(허용/거부)할 수 있습니다.

4. 사진 및 이미지 데이터의 처리

① 회원이 피드에 업로드한 사진은 Supabase Storage(feed-images 버킷)에 저장됩니다.

② 업로드된 사진은 해당 모임 소속 회원에게만 공개되며, 모임 외부에서는 접근할 수 없습니다.

③ 회원이 게시글을 삭제하면 첨부된 사진도 함께 삭제됩니다.

④ 회원 탈퇴 시 해당 회원이 업로드한 모든 사진은 삭제됩니다.

⑤ 사진 업로드 시 이미지는 서비스 서버를 통해 압축·리사이즈 처리될 수 있으며, 원본 EXIF 메타데이터(위치 정보 등)는 제거됩니다.

5. 개인정보의 보유 및 파기

① 회원 탈퇴 시 즉시 파기합니다. 이에는 업로드된 사진, 게시글, 댓글, 투표 데이터가 포함됩니다.

② 관련 법령에 의해 보존이 필요한 경우 해당 기간 동안 보관합니다:

  • 서비스 이용 기록: 3개월 (통신비밀보호법)

③ 파기 방법: 전자적 파일은 복구 불가능한 방법으로 영구 삭제하며, 클라우드 스토리지에 저장된 사진·이미지 파일도 동일하게 삭제합니다.

6. 개인정보의 제3자 제공

서비스는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 이용자의 동의가 있거나 법률에 특별한 규정이 있는 경우에 한해 제공합니다.

같은 모임에 소속된 회원 간에는 다음 정보가 공유됩니다:

  • 닉네임, 이름, 직책, 모임 가입일, 성별, 생년월일
  • 회비 납부 여부
  • 피드 게시글(텍스트 및 사진), 댓글, 투표 항목 및 참여 결과, 좋아요·반응

7. 개인정보의 국외 이전

이전 받는 자이전 목적이전 항목이전 국가보유 기간
Supabase Inc.클라우드 데이터 저장·인증, 사진 파일 저장이메일, 닉네임, 프로필 정보, 모임 데이터, 피드 게시글·댓글·투표 데이터, 업로드된 사진·이미지 파일미국(AWS)회원 탈퇴 시 즉시 삭제
Google LLC (Firebase)푸시 알림 발송기기 토큰(FCM), OS 정보미국로그아웃 또는 탈퇴 시 즉시 삭제

8. 이용자의 권리와 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 개인정보 열람·수정: 앱 내 프로필 수정 (설정 → 내 프로필)
  • 회원 탈퇴 및 개인정보 삭제: 앱 내 설정 → 회원 탈퇴
  • 게시글·댓글·사진 삭제: 본인이 작성한 게시글, 댓글, 업로드한 사진을 앱 내에서 직접 삭제
  • 카메라·사진 라이브러리 접근 권한 철회: 기기 설정에서 권한 변경
  • 푸시 알림 수신 거부: 기기 설정에서 알림 끄기
  • 개인정보 처리 정지 요청: 아래 연락처로 요청

9. 만 14세 미만 아동의 개인정보

서비스는 만 14세 미만 아동의 회원 가입을 제한합니다. 만 14세 미만임이 확인된 경우 즉시 개인정보를 파기합니다.

10. 개인정보의 안전성 확보 조치

  • 데이터 암호화: 전송 시 SSL/TLS 적용, 저장 시 암호화
  • 접근 통제: 인증된 사용자만 본인 데이터 접근 가능 (Row Level Security)
  • 사진 접근 통제: 업로드된 사진은 해당 모임 소속 회원만 열람 가능하도록 스토리지 정책 적용
  • 비밀번호 미저장: 소셜 로그인만 사용하여 비밀번호를 수집·저장하지 않음
  • FCM 토큰 관리: 로그아웃·탈퇴 시 기기 토큰 즉시 삭제
